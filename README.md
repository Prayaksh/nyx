# Nyx

Nyx is a terminal-first AI agent built on Bun that combines conversational AI, planning workflows, and autonomous task execution in a single CLI.

It provides multiple operating modes ranging from simple question-answering to full agentic execution, while maintaining a clean terminal experience and support for external integrations.

---

## Features

### Agent Mode

Execute tasks through an AI agent capable of reasoning, planning, and taking actions through available tools.

### Ask Mode

Lightweight conversational mode for quick questions and information retrieval.

### Plan Mode

Generate structured execution plans before running tasks, allowing you to review the approach before any action is taken.

### Terminal Interface

A dedicated terminal experience with:

- Markdown rendering
- Interactive prompts
- Action tracking
- Diff visualization
- Session-based workflows

### Extensible Integrations

Nyx includes infrastructure for external communication channels such as Telegram and WhatsApp, enabling the same workflows outside the terminal.

## Installation

Clone the repository:

```bash
git clone https://github.com/Prayaksh/nyx.git
cd nyx
```

Install dependencies:

```bash
bun install
```

Create a `.env` file using the provided example:

```bash
cp .env.example .env
```

Configure the required environment variables:

```env
OPENROUTER_API_KEY=
OPENROUTER_DEFAULT_MODEL=openrouter/free
FIRECRAWL_API_KEY=
```

---

## Local CLI Setup

To use `nyx` as a global command during development, ensure your `package.json` contains:

```json
{
  "bin": {
    "nyx": "index.ts"
  }
}
```

Link the package globally:

```bash
bun link
```

You can then run:

```bash
nyx
```

from any directory on your system.

## Commands

### Initial Setup

```bash
nyx wakeup
```

Runs the configuration wizard and initializes local settings.

### Agent Mode

```bash
nyx agent
```

Starts the autonomous agent workflow.

### Ask Mode

```bash
nyx ask
```

Starts a lightweight conversational session.

### Plan Mode

```bash
nyx plan
```

Generates structured plans before execution.
